Definition
The Market Identifier Code (MIC) is a unique code used to identify securities trading exchanges, regulated and non-regulated trading markets.

Additional Information
The MIC is a four alpha character code, and is defined in ISO 10383 by International Organization for Standardization. For example, trades that are executed on the NASDAQ market are identified using MIC code XNAS.
