Definition
The CUSIP-linked MIP code (CLIP), RED CLIP or RED Code is used to identify the reference entity of a credit default swap. It is mainly used as a key field in Markit's Reference Entity Database (RED). Each CLIP is linked with one or more CUSIPs each representing reference entity obligations.

Additional Information
MIP is a synonym of Markit Partners. S&P and Markit Partners introduced the CLIP concept. Entity CLIP codes are generated by the S&P CUSIP Bureau and are assigned to each entity name individually. CLIP codes are used for electronic matching through the Depository Trust & Clearing Corporation (DTCC) and have been integrated in various online platforms.
