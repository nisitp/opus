Security

Status
Public

Definition
The VALOR number is a code which uniquely identifies listed securities and financial instruments traded in Switzerland, Liechtenstein and Belgium.

Additional Information
A VALOR number can be used in conjunction with the Market Identifier Code and the Currency Code to uniquely identify a traded instrument. It can be used in transaction reporting and for position keeping.
