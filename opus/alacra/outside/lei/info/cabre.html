Definition
The CUSIP Avox Business Reference Entity Identifier. or CABRE. is a combined effort by CUSIP Global Services and Depository Trust & Clearing Corporation/Avox to provide a universal identification system for global business entities.

Additional Information
The CABRE identifier is a 10-character code that contains the 2-character ISO domicile code based on the location of the entity and embeds the CUSIP issuer number, when available.
